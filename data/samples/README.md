# AI Knowledge Search Platform - Sample Documents

This directory contains sample documents for demonstrating the AI Knowledge Search Platform.

## Document Categories

### Technical Documentation
- `ai_basics.md` - Introduction to artificial intelligence
- `machine_learning_guide.md` - Comprehensive guide to machine learning
- `neural_networks.md` - Deep dive into neural networks
- `deep_learning.md` - Advanced deep learning concepts

### Research Papers
- `transformer_paper.md` - Attention is All You Need (simplified)
- `bert_paper.md` - BERT: Pre-training of Deep Bidirectional Transformers
- `gpt_paper.md` - Language Models are Few-Shot Learners

### Business Reports
- `ai_industry_report.md` - AI industry trends and analysis
- `ml_adoption_report.md` - Machine learning adoption in enterprises
- `ai_ethics_report.md` - Ethical considerations in AI development

### FAQs and Help
- `ai_faq.md` - Frequently asked questions about AI
- `ml_troubleshooting.md` - Common machine learning problems and solutions
- `getting_started_guide.md` - Getting started with AI development

## Usage

To ingest these sample documents into the system:

```bash
python scripts/ingest_samples.py
```

This will process all documents in this directory and make them available for search.
